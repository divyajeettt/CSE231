cat

NAME
        cat - concatenate files and print on the standard output.

SYNOPSIS
        cat [OPTION]... [FILE]...

DESCRIPTION
        Concatenates files to the standard output. When no file is specified, standard input is taken by default. When multiple files are given as arguments, displays the contents of all of them inn order.

SUPPORTED OPTIONS:
        -n (number)
            enumerate all output lines

        -E (show-ends)
            display a $ symbol at the end of each line
