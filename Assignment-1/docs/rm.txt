rm

NAME
        rm - remove files or directories from the system.

SYNOPSIS
        rm [OPTION(s)] [FILE(s)/DIRECTORY(ies)]

DESCRIPTION
        Removes file from the system. By default, it cannot remove directories.

SUPPORTED OPTIONS:
        -r (recursive)
            recursively remove directories and their contents

        -v (verbose)
            explain what is being done, i.e., print a message for each removed file/directory
