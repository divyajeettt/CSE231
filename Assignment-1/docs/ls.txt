

ls

NAME
        ls - list down the files and directories in the current directory

SYNOPSIS
        ls [OPTION(s)] [DIRECTORY(ies)]

DESCRIPTION
        Lists down the names of files and directories in a directory. By default, lists downs the items in the current directory. Unlike the Artix Shell, the list is not sorted by default.

SUPPORTED OPTIONS
        -a (all)
            display all files and directories, i.e., do not ignore the items starting with .

        -1
            display only one file per line, i.e., display a newline-separated list of files and directories

ASSUMPTIONS
        The output list is not sorted by default.
